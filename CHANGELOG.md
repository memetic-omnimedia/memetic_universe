# Changelog

## [Unreleased]

### Added
- Comic title header with "Memetic Universe" branding and issue number
- Responsive layout with mobile breakpoint at 700px
- Comic-style typography using Comic Neue and Bangers fonts from Google Fonts
- Variable panel spacing classes (pace-tight, pace-wide) for pacing control
- Subtle paper-like background texture
- Footer with "To Be Continued..." end marker
- Viewport and charset meta tags
- DESIGN_CHANGES.md documenting all visual design decisions

### Changed
- Background color from flat #f0f0f0 to warmer #e8e4df with noise texture
- Speech bubble tails now only appear on the first box in multi-box panels
- Font size increased from 14px to 15px, line height from 1.5 to 1.6
- Panel images now have a subtle border for better definition
